- Added Cloudflare KV caching on the worker with a 6-hour cron auto-refresh

- Added localStorage cache (6h 15m TTL) to prevent redundant worker requests on page reload
- Added isFetching guard to block concurrent fetch calls
- Auto-refresh interval aligned to 6h 15m to give cron a 15-minute buffer before client pulls
- Fixed duplicate style attribute bug on card elements (animation-delay was being dropped)
